# Установка RAGRAF на Windows для аналитика заказчика

> RAGRAF — это локальное ПО для проектирования цифрового двойника управленческих процессов.
> Все данные хранятся на компьютере аналитика. Никакие данные не уходят за пределы
> вашей машины.

## 🟢 Быстрый старт (5 минут)

### 1. Скачайте установщик

С главной страницы [RAGRAF на GitHub](https://github.com/Barbashin1970/RAGRAF) или сайта
проекта скачайте файл **`start-ragraf.bat`** в любую папку (например, на рабочий стол).

### 2. Установите 3 программы (один раз)

RAGRAF требует три бесплатные программы:

| Программа | Зачем | Где скачать |
|---|---|---|
| **Git** | для скачивания и обновления RAGRAF | https://git-scm.com/download/win |
| **Python 3.11+** | для backend-сервера | https://www.python.org/downloads/ |
| **Node.js 18+** | для frontend-сборки | https://nodejs.org/ |

**Важно:** при установке Python поставьте галочку **«Add Python to PATH»** —
иначе скрипт не найдёт `python` в командной строке.

### 3. Запустите `start-ragraf.bat`

Двойной клик → откроется чёрное консольное окно. Скрипт:

1. Проверит, что Git / Python / Node установлены.
2. Скачает RAGRAF в `C:\Users\<ваше_имя>\RAGRAF\` (если ещё нет).
3. Если RAGRAF уже скачан — проверит обновления и спросит «Обновить?».
4. Установит зависимости (1-5 минут первый раз).
5. Соберёт фронтенд (1-2 минуты).
6. Запустит сервер в отдельном окне.
7. Откроет браузер на http://127.0.0.1:8080.

**Первый запуск** занимает 5-10 минут (зависит от интернета).
**Последующие запуски** — 10-20 секунд.

### 4. Работайте в браузере

После открытия http://127.0.0.1:8080 — вы видите **тёплый бежевый интерфейс**
с навигацией: Студия аналитика, Регламенты, Датчики, Модули, Аудит,
Граф связей, Цифровой двойник.

Все данные пишутся в `C:\Users\<ваше_имя>\RAGRAF\data\`. Бэкапить — копируйте
эту папку.

### 5. Чтобы остановить

Закройте окно с заголовком **«Windows PowerShell»** (где видны логи uvicorn).

## 🔄 Обновление

При очередном запуске `start-ragraf.bat` скрипт сам проверит наличие обновлений
на GitHub:

```
==> Проверка обновлений
    [!] Доступна новая версия: a1b2c3d4 → e5f6g7h8
        * фикс 048 — UI аудит-журнала
        * фикс 047 — sensor pill без subtype
        * фикс 046 — паспорт прикладных модулей
    Обновить локальную копию до e5f6g7h8? (Y/N) [Y]:
```

Нажмите **Enter** (или Y) — Git подтянет последний коммит и пересоберёт фронтенд.
Ваши данные в `data/` **не затрагиваются** при обновлении кода.

Если хотите остаться на старой версии — нажмите N. Платформа запустится на
текущей версии.

## 📁 Что и где хранится

```
C:\Users\<ваше_имя>\RAGRAF\          ← корень установки
├── backend\                          ← Python-сервер
│   └── .venv\                        ← виртуальное окружение Python
├── frontend\                         ← React-фронт
│   ├── node_modules\                 ← Node-зависимости
│   └── dist\                         ← собранная SPA (отдаётся backend'ом)
├── data\                             ← ВАШИ ДАННЫЕ (бэкапьте отсюда)
│   ├── regulations.duckdb            ← регламенты + аудит-журнал
│   ├── fixtures\                     ← seed-фикстуры (можно править)
│   ├── flows\                        ← flow.json регламентов
│   └── versions\                     ← история версий flow
└── _run-server.ps1                   ← скрипт запуска сервера
```

## 🛠 Что делать если что-то не работает

### «git не установлен»
Скачайте Git с https://git-scm.com/download/win и установите со всеми
настройками по умолчанию. Перезапустите `start-ragraf.bat`.

### «python не установлен» или «Python 3.10»
Установите свежий Python 3.11+ с https://www.python.org/downloads/.
**ОБЯЗАТЕЛЬНО** поставьте галочку «Add Python to PATH» в первом окне установщика.

### «node не установлен»
Скачайте Node.js LTS (18+) с https://nodejs.org/ — выберите Windows installer.

### Backend не запускается / порт 8080 занят
Откройте `C:\Users\<имя>\RAGRAF\_run-server.ps1` в Блокноте, замените `--port 8080`
на другой свободный порт (например 8090). Запустите вручную в PowerShell.

### Хочу полностью переустановить
Удалите папку `C:\Users\<имя>\RAGRAF\` (включая `data\` — это сбросит и данные)
и запустите `start-ragraf.bat` заново.

### Хочу сохранить данные, но переустановить код
1. Скопируйте `C:\Users\<имя>\RAGRAF\data\` в безопасное место.
2. Удалите `C:\Users\<имя>\RAGRAF\`.
3. Запустите `start-ragraf.bat` — он склонирует свежую версию.
4. Скопируйте сохранённый `data\` обратно в `C:\Users\<имя>\RAGRAF\data\`.

## 🔐 Безопасность

- Платформа слушает только на **127.0.0.1** (loopback) — снаружи компьютера
  недоступна. Если нужен сетевой доступ для коллег, поменяйте `$BackendHost`
  в `ragraf.ps1` на `0.0.0.0` (на свой страх).
- Никакие данные не отправляются за пределы вашего компьютера. Платформа
  работает **offline** после первого скачивания зависимостей.
- Лицензия указана в репозитории на GitHub. По всем вопросам интеграции —
  обращайтесь к авторам.
