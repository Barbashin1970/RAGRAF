# Датчик давления (`p`)

Манометр или преобразователь давления на трубопроводе. На теплосетях
ставится на подаче, обратке, перед/после задвижек, рядом с насосами.

## Прибор

- Электронный манометр / piezo-преобразователь
- Аналоговый сигнал 4–20 мА конвертируется в число edge-устройством
- Точность ±0.5% от шкалы, дискретизация 1–10 раз/сек

## Поля payload

| Поле          | Тип    | Ед.   | Описание                                                |
|---------------|--------|-------|---------------------------------------------------------|
| `pressure`    | number | атм   | Текущее показание манометра                              |
| `reference`   | number | атм   | (опционально) проектное / эталонное значение для узла   |
| `x`, `y`      | number | м     | Координаты точки замера на схеме сети                    |
| `edge_id`     | number | —     | (опционально) ID участка трубопровода                    |
| `temperature` | number | °C    | (опционально) совмещённое измерение температуры (мульти-датчик) |

## Типичные диапазоны

- Холодное водоснабжение: 2–6 атм
- Горячее водоснабжение / отопление: 4–10 атм
- Магистральные трубопроводы: до 25 атм
