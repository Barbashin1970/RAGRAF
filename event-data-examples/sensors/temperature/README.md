# Датчик температуры (`t`)

Термопара (TC) или термосопротивление (RTD, PT100/PT1000). На теплосетях
ставится на подаче и обратке, в баках, на двигателях насосов.

## Прибор

- TC type K (0..1300 °C) или PT100 (-200..600 °C)
- 4-проводное подключение для точности
- Опрос 1–60 сек

## Поля payload

| Поле           | Тип    | Ед.   | Описание                                       |
|----------------|--------|-------|------------------------------------------------|
| `temperature`  | number | °C    | Текущая температура                             |
| `reference`    | number | °C    | (опционально) проектная температура              |
| `x`, `y`       | number | м     | Координаты точки замера                          |
| `location`     | string | —     | (опционально) `supply` / `return` / `tank` …    |
| `edge_id`      | number | —     | ID участка                                       |

## Типичные диапазоны

- Холодная вода: 5..15 °C
- Горячая вода: 55..75 °C
- Отопление, подача: 70..130 °C (зависит от температурного графика)
- Отопление, обратка: 40..70 °C
