# Расходомер (`flow`)

Электромагнитный или ультразвуковой расходомер на трубопроводе. Меряет
объёмный расход теплоносителя или воды — ключевой параметр для теплоучёта
и обнаружения утечек.

## Прибор

- Электромагнитный: для чистой воды без воздуха, точность ±0.5%
- Ультразвуковой: накладной (clamp-on), для любых сред, точность ±1..2%
- Опрос 1–10 сек, интеграция по времени → объём м³

## Поля payload

| Поле          | Тип    | Ед.   | Описание                              |
|---------------|--------|-------|---------------------------------------|
| `flow`        | number | м³/ч  | Текущий расход                         |
| `reference`   | number | м³/ч  | (опционально) проектное значение       |
| `total_volume`| number | м³    | (опционально) накопленный объём        |
| `direction`   | string | —     | (опционально) `forward` / `reverse`    |
| `x`, `y`      | number | м     | Координаты точки замера                |
| `edge_id`     | number | —     | ID участка                              |

## Типичные диапазоны

- ИТП жилого дома (60–100 квартир): 5..15 м³/ч
- ЦТП квартала: 50..200 м³/ч
- Магистральный трубопровод: 500..2000 м³/ч
